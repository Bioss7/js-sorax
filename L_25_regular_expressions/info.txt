// Регулярным выражение называется объект который описывает набор символов или символьный шаблон, 
// который можно использовать для поиска подстрок в каком либо тексте

/g - глобальный поиск ищет все совпадения, не глобальный, первое.
/i - insensitiveCase поиск не чувствительный к регистру
/m - multiline
их можно комбинировать
/gi
Вообще регулярное выражение имеет такой синтаксис
/RegEx/flags
[aoeuiy]//найдет все гласный
[a-z]//найдет все буквы от а до z (диапазон) здесь могут быть буквы,цифры
//и не обязательно все
/\d/g // найти все цифры
/\D/g // найти все кроме цифр
/\w/g // найти все буквы и цифры, нижние подчеркивания
/\W/g // найти все что ими не является(в тч пробелы, дефисы, точки запятые)
/\s/g // найти все пробелы табуляции переносы строк
/\S/g // найти все что ими не является
/an/g // найти все буквосочетания an где бы они не встретились
/\ban\b/gi // найти артикль an отделенный с двух сторон любыми \W символами
/\Ban\B/gi // найти все буквосочетания an у которых с обоих сторон \w
/an\B/gi // найти все буквосочетания an у которых слева \W|\w справа\w
//по русски говоря все что не кончается на an
/\Ban/gi // все что не кончается на an
/\bme\b|\band\b/gi //найти все слова me или and
/gr(e|a)y/gi // все gray grey преимущество этого способа что можно применять к 
//словам где могут отличаться не одна а несколько букв
/colou?r/gi // все color colour - буква перед ? является не обязательной
/./gi// любой символ кроме переноса строки, чтобы найти точку экранируем \.
// это касается любых спец.символов задействованных в RegEx
/a....e/gi // любые буквосочетания где между а и е 4 любых символа
/a.{4}e/gi // то же что выше
/a.{2,6}e/gi // жадное выражение где найдутся любые буквосочетания где между 
// а и е от 2 до 6 любых символов причем предпочтение будет тем у кого их больше
/a.{2,6}?e/gi // не жадное(приоритет к меньшему числу .к)
/a.{1,}?e/gi // не жадное от 1 до "Бесконечность не предел!" есть альтернатива:
/a.+?e/gi 
/a.*?e/gi // match даже если а и е стоят рядом
/^that/gim // match для всех that в начале строки.
/\.$/gim // match для всех . в конце строки.
// Впередсмотрящее утверждение и впередсмотрящее отрицание (pos/neg lookahead)
/\b\w+?\b(?=[\s,]*?and)/gi // слово нной длины(без дефисов) за которым от 0 до 
// бесконечности пробельных символов и запятых, после которых идет and(positive)
/\bto\b(?! know)/ // все to кроме тех за которыми идет know(negative lookahead)
/(\b\w+)@(\w+\.\w+\b)/ // все мыла разделенные на две группы до @ и после
/java(?:script)? // найти все java и javascript не создавая запоминающие группы
